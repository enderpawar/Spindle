# Phase 실행

절차의 단일 원본은 **`PLAN.md`의 "실행 절차 (모든 에이전트 공통)" 절**이다. 이 스킬은 진입점일 뿐이며, 절차를 여기서 재정의하지 않는다.

1. `PLAN.md`를 연다.
2. 인자가 숫자 N이면 "실행 절차" 1~5단계를 그대로 수행한다.
3. 인자가 없거나 `status`면 진행 상태만 보고한다.
